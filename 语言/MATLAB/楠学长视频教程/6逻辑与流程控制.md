### 循环结构

#### for循环结构
`步长默认值为1, 可以省略`
```
for 循环变量 = 初值:步长:终值
  执行语句1;
  ...
  执行语句n;
end
```

#### while循环
```
while 条件表达式
  执行语句1;
  ...
  执行语句n;
end
```

### 分支结构
#### if...end结构
```
if 条件表达式
  语句体;
end
```

#### if...else...end结构
```
if 表达式
  语句体1;
else 
  语句体2;
end
```

#### switch...case...end结构
```
switch 表达式(数值或字符串)
  case 数值或字符串1
      语句体1;
  case 数值或字符串2
      语句体2;
  ...
  otherwise
      语句体n;
end
```
